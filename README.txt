# Учебный материал по разработке модов для Minecraft Forge (1.20.1)
Данный репозиторий представляет собой обучающий ресурс для начинающих моддеров Minecraft. Здесь вы найдете примеры кода, шаблоны проектов и инструкции для освоения модификации игры с использованием Minecraft Forge.

Цель проекта:
- Познакомить с основами работы с Forge MDK
- Показать лучшие практики структурирования модов
- Объяснить ключевые аспекты взаимодействия с API Minecraft
- Предоставить готовую среду для экспериментов и обучения

---

## Настройка среды разработки
Проект использует стандартную методологию установки Minecraft Forge. Для начала работы выполните следующие шаги:

### Процесс установки:
1. **Клонируйте репозиторий** в выбранную директорию
2. Откройте терминал в корневой папке проекта

Выберите предпочитаемую IDE:

#### Для Eclipse:
```bash
./gradlew genEclipseRuns
